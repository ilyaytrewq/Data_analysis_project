# Telco Customer Churn — Исследовательский проект

Проект по курсу «Анализ данных в Python» (НИС). Анализ оттока клиентов телекоммуникационной компании.

**Авторы:** Тихонов Илья, Тахунов Арсен

## Датасет

[Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn) (IBM Sample Data Sets) — 7 043 клиента, 21 признак: демография, подключённые услуги, тип контракта, платежи, факт оттока (Churn).

Файл: `WA_Fn-UseC_-Telco-Customer-Churn.csv`

## Ключевые результаты

- **Главные факторы оттока:** помесячный контракт, оптоволоконный интернет, оплата электронным чеком, короткий стаж обслуживания.
- **Три сегмента клиентов:** «новички с дешёвым тарифом», «новички с дорогим тарифом» (группа риска), «лояльные долгожители».
- Модель логистической регрессии позволяет предсказывать отток с приемлемой точностью.
